# Privacy Policy

Offiqa is designed as a local-first browser extension.

## Data storage

Offiqa stores user data locally in the browser using Chrome extension storage and IndexedDB. This may include notes, workspace data, quick links, focus timer state, preferences, and other extension settings.

## No account required

Offiqa does not require users to create an account.

## No selling of data

Offiqa does not sell user data.

## No third-party tracking

Offiqa does not include third-party advertising trackers.

## Browser permissions

Offiqa may request browser permissions to support features such as new tab customization, quick notes, workspace/session restoration, bookmarks, reminders, and local productivity tools.

## Clipboard access

Clipboard-related permissions are used only for user-facing productivity features. Offiqa does not upload clipboard content to a remote server.

## History, tabs, and bookmarks

History, tabs, and bookmarks permissions are used to power local browser productivity features. Offiqa does not sell or share this data.

## Contact

For privacy questions, please contact the project maintainer through GitHub Issues.
