# Offiqa

Offiqa is a calm, practical browser new tab workspace for office workers.

## Features

- New Tab workspace
- Quick work apps
- Continue workspace
- Quick notes
- Focus timer
- Smart reminders
- Role-based work blocks for office, sales, marketing, design, development, and freelancers

## Install for development

1. Clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable Developer mode.
4. Click Load unpacked.
5. Select the project folder.
## Privacy-first, local-first

Offiqa is designed to work locally in your browser. Notes, workspace data, timer state, quick links, and preferences are stored locally.

Offiqa does not sell user data and does not require an account.

## Browser permissions

Offiqa requests browser permissions to support features such as:

- New Tab replacement
- Quick notes
- Workspace/session restoration
- Bookmarks and quick access
- Focus timer and reminders
- Local productivity tools

Some permissions may look broad because browser extensions require them for features that work across tabs and pages.
