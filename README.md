# Offiqa

Offiqa is a calm, practical browser new tab workspace for office workers.

## Features

- New Tab workspace
- Quick work apps
- Continue workspace
- Quick notes
- Focus timer
- Smart reminders
- Role-based work blocks for office, sales, marketing, design, development, and freelancers

## Install for development

1. Clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable Developer mode.
4. Click Load unpacked.
5. Select the project folder.
