# Security Policy

## Reporting a vulnerability

If you discover a security issue in Offiqa, please open a GitHub Issue with clear reproduction steps.

Please do not include sensitive personal data in public issues.

## Scope

Security reports may include:

- Extension permission issues
- Local data exposure
- Clipboard-related issues
- Unsafe content script behavior
- Cross-site scripting risks
- Data loss bugs

## Local-first design

Offiqa is designed to keep user data local whenever possible.
